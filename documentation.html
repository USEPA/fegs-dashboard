The Whole Process' Synopsis

The tool fegs-dashboard.xlsm gathers, records, and analyzes ratings of a site's natural features for various people. The rating process is described in detail below. The basic usage of this tool is to open fegs-dashboard.xlsm and be users through the workflow, downward from the top of the worksheet named dash.

Structure of Entered Data

Ratings are stored in data structured to give each record the rating of a single attribute for a single beneficiary. For example, say a rating of poor is submitted for boaters listing several natural features. Upon submission, the tool creates an individual record for each natural feature listed in this rating. The Whole Process Enumerated by Step

  1. Identify the site.

    2. Type the name of the site being rated in the text box.

  2. Identify beneficiaries.

    3. Identify all beneficiaries of the site.

  3. Choose a single beneficiary from those identified in step two.

    4. Choose a single beneficiary to be associated with this rating.

  4. List attributes of the site to recieve the current rating.

    5. List attributes to be rated by the current rating.

  5. Rate the satisfaction level of this beneficiary.

    6. Rate how satisfied the chosen beneficiary is with the attributes identified.

  6. Explain this rating. EG, what influenced the rating?

    7. Provide supplemental information about the rating. Indicate whether surrounding infrastructure affected the rating with the checkbox.

  7. Submit the data.

    8. Use the submit button to submit the data from the dash worksheet to the data worksheet.

  8. Repeat steps 3-7 for remaining beneficiaries.

    9. Use the clear data button to clear the data on the dash worksheet.

  9. Analyze the data.

    10. Use the analyze button to generate an editable pivot table and pivot chart. The pivot table is layed out so that it initially show the percentage of poor, fair, and good ratings for each pair of attribute

  10. Save the workbook to keep the data entered.

    11. The Excel document must be saved to save the data entered.
