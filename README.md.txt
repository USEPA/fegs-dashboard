# fegs-dashboard
The fegs-dashboard provides a measure of goods and services by surveying users of the tool of qualitative data. An example of the data users are asked is "How would you rate the water quantity in your region's rivers and streams?"

## prototyping features within prototype_3.xlsx
The file prototype_3.xlsx is the pretty proof of concept. The file is to be used to interactively fill out a survey.
